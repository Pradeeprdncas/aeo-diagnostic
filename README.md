# AEO Diagnostic

AI Visibility Analytics Tool

## What it does
Analyzes brand visibility across multiple AI models (Groq, Mistral, OpenRouter)

## Features
- Multi-model comparison
- Visibility scoring system
- AI agreement metric
- Insight generation
- Report card system

## Tech Stack
- React
- Node.js / Express
- AI APIs (Groq, Mistral, OpenRouter)
